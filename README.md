# Workfolio

## Prerequisites
<ol>
    <li> npm
    <li> npx
    <li> Tailwind
    <li> Typescript
</ol>

## Building project
### Build Tailwind stuff
```
npx tailwindcss -i ./src/input.css -o ./dist/output.css
```
For development use ```--watch```.

### Build Typescript stuff
```
npx webpack
```
For development use ```--watch```.
